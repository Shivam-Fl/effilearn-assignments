# effilearn-assignments

This repository is used by Effilearn students to submit assignments and practice using Git and GitHub.

## Purpose

- Submit coursework and assignments for review.
- Practice core Git workflows: cloning, branching, committing, and pushing changes.
- Practice collaboration workflows: forking, opening pull requests, and resolving merge conflicts.

## Getting Started

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd effilearn-assignments
   ```
2. Create a new branch for your assignment:
   ```bash
   git checkout -b <your-name>/<assignment-name>
   ```
3. Add your work, then commit and push:
   ```bash
   git add .
   git commit -m "Add solution for <assignment-name>"
   git push origin <your-name>/<assignment-name>
   ```
4. Open a pull request for review.

## Guidelines

- Keep each assignment in its own folder, named after the assignment.
- Write clear, descriptive commit messages.
- Do not modify or delete other students' submissions.
